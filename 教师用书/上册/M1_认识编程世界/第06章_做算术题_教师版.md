## 🗺️ 教学流程

### 第 1 课时：基本运算 + 整数除法（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-5 分 | 导入 | "17 颗糖分给 5 个小朋友"故事 |
| 5-15 分 | 讲授 | 七个运算符一览。重点对比 / 和 %："一个告诉你每人分几颗，一个告诉你剩几颗" |
| 15-30 分 | 演示 | **整数除法实验**：让学生依次运行 `5/2`、`5.0/2`、`double x = 5/2;`，每次猜结果再验证。这个错误会在后续无数次出现，必须在本章彻底打透 |
| 30-45 分 | 板书 | 在黑板上画"操作数类型决定除法类型"的决策树。整数/整数 = 整数除法，其他 = 浮点除法 |

### 第 2 课时：取模与自增（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-10 分 | 复习 | / 和 % 回顾 |
| 10-30 分 | 讲授 | 取模三大用法：①判断奇偶（口头解释——余数为 0 即偶数，暂不写 if 代码） ②循环到头 ③提取数位。每个用法配一个生活例子和一个代码示例 |
| 30-40 分 | 讲授 | ++ / -- / 前置后置区别。**建议学生现阶段只用后置（a++）独占一行**，避免过早玩花样 |
| 40-45 分 | 演示 | 展示 `int b = a++;` vs `int b = ++a;` 的区别——两行代码，对照输出 |

### 第 3 课时：复合赋值 + 优先级（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-15 分 | 讲授 | +=、-=、*=、/=、%=。每个用"展开版"vs"简写版"对比展示 |
| 15-30 分 | 讲授 | 优先级。板书优先级金字塔。重点原则：**记不住就加括号**——这不是偷懒，这是专业做法 |
| 30-45 分 | 挑战 | 给几个需要括号的表达式，让学生加括号。比如 `a * b + c / d` 想先算 b+c → `a * (b + c) / d` |

### 第 4-5 课时：练习 + 小测（90 分钟）

重点检查整数除法的陷阱是否被击穿。

## 🔥 重点难点

### 重点

1. **整数除法丢弃小数**：最容易犯的错，没有之一。通过大量动手实验形成条件反射
2. **取模的用途**：不仅要会算余数，要知道它真正有用的场景
3. **括号 > 优先级**：实用主义编程思维

### 突破策略

| 卡点 | 突破 |
|------|------|
| 5/2 = 2 总忘了 | "计算机不是数学老师——它不会帮你保留小数。你让它做整数除法它就丢掉小数，一丁点都不留" |
| ++a vs a++ | 画时间线：a 初始值 5。`b = a++` → t1: b=5, t2: a=6。`c = ++a` → t1: a=7, t2: c=7 |
| 复合赋值 `x *= y + 1` | 强调"复合赋值先把右边整个算完"——`y+1` 先算，再用结果乘 x。用括号思维理解：`x = x * (y + 1)` |

## 💬 常见问答

**Q1："取模有什么用啊？我考试从没用过余数。"**

A：编程里取模是超级工具。判断奇偶、循环计数、取数字的某一位、哈希算法、加密解密……全都要用到取模。等你学到后面就会发现："怎么又是 %？"

**Q2："自增自减为什么要分前后，搞这么复杂？"**

A：因为它可以一行做完两件事（用值 + 加 1）。不过现阶段你不用纠结——所有 `a++` 都独占一行就行。前后区别以后用到了自然就懂了。

**Q3："优先级那么多，真的要全背吗？"**

A：不需要。记住三条：①括号最高 ②乘除高于加减 ③赋值最低。剩下的——加括号。

## 🎲 课堂活动

### 活动：取模转盘

**道具**：白板画一个圆盘，标 0-6（星期天到星期六）

**规则**：
- 假设今天是星期 3
- 老师喊"30 天后"，学生算 (3+30) % 7 = ?
- 老师喊"100 天后"，学生心算
- 第一个喊出正确答案的人得分
- 玩完问学生：没有取模的话，你怎么算？

**目的**：体会取模的实际价值。


### 提高题

**3. 三位数拆解（参考代码）**

```cpp
int n;
cin >> n;
cout << "百位：" << n / 100 << endl;
cout << "十位：" << (n / 10) % 10 << endl;
cout << "个位：" << n % 10 << endl;
```

### 挑战题

**5. 数字反转（参考代码）**

```cpp
int n;
cin >> n;
int ge = n % 10;
int shi = (n / 10) % 10;
int bai = n / 100;
int rev = ge * 100 + shi * 10 + bai;
cout << rev << endl;
```

**6. 平均速度**

去程时间：6/12 = 0.5h，回程时间：6/20 = 0.3h  
总时间 = 0.8h，总路程 = 12km  
平均速度 = 12/0.8 = 15 km/h  
（不是 (12+20)/2 = 16，因为两段时间不同。）

\`\`\`cpp
double t1 = 6.0 / 12;  // 0.5
double t2 = 6.0 / 20;  // 0.3
double avg = 12.0 / (t1 + t2);  // 15.00
cout << fixed << setprecision(2) << avg << endl;
\`\`\`

## 📦 补充题库

1. 输入一个四位数，分别输出千位、百位、十位、个位。（`n/1000`, `(n/100)%10`, `(n/10)%10`, `n%10`）

2. `int a = 2, b = 3; int c = a++ * ++b;` 求 a、b、c。（a=3, b=4, c=8 —— a++ 用 2，++b 用 4，2×4=8 赋给 c，然后 a 加 1）
