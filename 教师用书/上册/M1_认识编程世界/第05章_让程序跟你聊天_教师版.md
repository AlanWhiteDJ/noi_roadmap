## 🔥 重点难点

### 重点

1. **cin >> 和 cout << 的方向**：不可逆。让肢体动作参与记忆
2. **fixed + setprecision**：必须成对使用，缺 fixed 效果完全不同
3. **#include 与功能的关系**：iostream → cin/cout，string → string 类型，iomanip → setprecision

### 卡点突破

| 卡点 | 突破 |
|------|------|
| 箭头方向混淆 | "箭头体操" + 口诀："cout <<：往外送；cin >>：往里拉" |
| setprecision 不加 fixed | 演示对比：`setprecision(2)` vs `fixed << setprecision(2)`——让学生亲眼看到区别 |
| cin 读 string 遇空格截断 | 现场演示，不回避。预告 getline（后面会学） |
| 忘了 `#include <string>` | 故意让学生忘一次，看报错→记住。比讲十遍管用 |

## 💬 常见问答

**Q1："endl 和 \n 到底用哪个？"**

A：日常用哪个都行。endl 多一个"刷新缓冲区"的操作——在需要立刻看到输出的时候（比如提示用户输入前）用 endl，纯输出数据用 \n。现阶段不用纠结，随便用。

**Q2："为什么有的工具需要 #include，有的不需要？"**

A：C++ 的设计哲学是"用啥拿啥，不浪费"。cout/cin 需要 `<iostream>`，string 需要 `<string>`，setprecision 需要 `<iomanip>`。以后你每学一个新工具，就记住它对应的 include。

**Q3："用户输入错了怎么办？比如要数字但我输了字母。"**

A：好问题，这涉及输入验证，后面会专门讲。现阶段假设用户"听话"，按要求输入。

## 🎲 课堂活动

### 活动：箭头体操（第 1 课时用）

**规则**：
- 全体起立
- 老师说"cout"，学生双臂向前推（数据出去）
- 老师说"cin"，学生双臂向内收（数据进来）
- 老师说"cout << age"，学生先推手、再喊"age 送出去"
- 交替加速，直到混乱大笑

**目的**：用身体记忆。这个方向感能管整本书。

## ✅ 习题详解

### 基础题

**1. 填空题**
（1）cin  
（2）fixed、setprecision  
（3）回车（Enter）

**2. 代码改错**

三处错误：
1. `cin << x` → 箭头方向反了，应为 `cin >> x`
2. 缺少 `#include <iomanip>`（用了 fixed 和 setprecision）
3. `cout << ... << x` 末尾缺分号

### 提高题

**3. 计算器（参考代码）**

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "请输入两个整数：";
    cin >> a >> b;
    cout << "和 = " << a + b << endl;
    cout << "差 = " << a - b << endl;
    cout << "积 = " << a * b << endl;
    cout << "商 = " << a / b << endl;
    cout << "余数 = " << a % b << endl;
    return 0;
}
```

**4. 自我介绍（参考代码）**

```cpp
#include <iostream>
#include <string>
#include <iomanip>
using namespace std;

int main() {
    string name, subject;
    int age;
    double height;
    
    cout << "姓名：";  cin >> name;
    cout << "年龄：";  cin >> age;
    cout << "身高（米）：";  cin >> height;
    cout << "最喜欢的科目：";  cin >> subject;
    
    cout << "我叫" << name << "，今年" << age << "岁，";
    cout << fixed << setprecision(2) << height << "米，";
    cout << "最喜欢" << subject << "。" << endl;
    return 0;
}
```

### 挑战题

**5. 时间转换（参考代码）**

```cpp
#include <iostream>
using namespace std;

int main() {
    int total;
    cout << "请输入秒数：";
    cin >> total;
    int h = total / 3600;
    int m = (total % 3600) / 60;
    int s = total % 60;
    cout << total << " 秒 = " << h << " 小时 " << m << " 分 " << s << " 秒" << endl;
    return 0;
}
```
