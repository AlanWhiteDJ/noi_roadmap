## 🧱 知识讲解

### 20.1  find：大海捞针

`find` 在一大段文字里找一小段文字（子串），返回**第一次出现的位置**（下标）。如果找不到，返回一个特殊值 `string::npos`。

```cpp
string s = "I love programming, love C++";
int pos = s.find("love");          // pos = 2（第一个 love 在位置 2）
cout << pos << endl;               // 2

pos = s.find("love", 5);           // 从下标 5 开始找
cout << pos << endl;               // 18（第二个 love 在位置 18）

pos = s.find("Python");
if (pos == string::npos) {
    cout << "没找到" << endl;       // 输出这个
}
```

> ❌ **常见踩坑——find 返回值判断用 -1：**
>
> ```cpp
> if (s.find("abc") == -1) { ... }   // ❌ 有些编译器能用，但不规范
> ```
>
> ✅ **正确——标准写法：**
>
> ```cpp
> if (s.find("abc") == string::npos) { ... }  // ✅ C++ 标准写法
> ```
>
> 🧠 `string::npos` 是一个特殊常量，表示"没有这个位置"（no position）。它的值通常是 -1，但直接用 `string::npos` 最规范。

**实战：统计某个词出现了几次**：

```cpp
string s = "apple banana apple orange apple";
string target = "apple";
int cnt = 0;
int pos = 0;

while ((pos = s.find(target, pos)) != string::npos) {
    cnt++;
    pos += target.size();           // 跳过已找到的，继续找下一个
}
cout << "apple 出现了 " << cnt << " 次" << endl;  // 3
```

### 20.2  substr：剪刀裁剪

`substr` 像一把剪刀，从字符串中间剪下一段。

```cpp
string s = "Hello World";

string sub1 = s.substr(0, 5);      // 从下标 0 开始，取 5 个 → "Hello"
string sub2 = s.substr(6, 5);      // 从下标 6 开始，取 5 个 → "World"
string sub3 = s.substr(6);         // 从下标 6 开始，取到末尾 → "World"
```

格式：`s.substr(起始位置, 长度)`。不写长度就取到末尾。

> ❌ **常见踩坑——substr 取的长度超出字符串长度：**
>
> ```cpp
> string s = "Hi";
> string sub = s.substr(0, 10);     // ❌ 长度 10 超过字符串，行为不确定
> ```
>
> ✅ **正确**：先判断长度够不够：
>
> ```cpp
> if (s.size() >= 10) {
>     string sub = s.substr(0, 10);
> }
> ```

**实战：判断一个字符串是不是另一个的前缀**：

```cpp
string s = "hello.cpp";
if (s.substr(0, 5) == "hello") {
    cout << "这是 C++ 源文件" << endl;
}
```

### 20.3  erase：橡皮擦

`erase` 从字符串中删除一段。

```cpp
string s = "Hello Beautiful World";

s.erase(5, 10);                    // 从下标 5 开始，删 10 个字符
cout << s << endl;                 // "Hello World"

s.erase(5);                        // 从下标 5 开始，删到末尾
cout << s << endl;                 // "Hello"

s.erase();                         // 全部删光
cout << s << endl;                 // ""
```

**实战：去掉字符串的首尾空格**（简化版——只去掉开头空格）：

```cpp
string s = "   Hello";
while (s.size() > 0 && s[0] == ' ') {
    s.erase(0, 1);                 // 删掉第 0 个字符（每次删 1 个）
}
cout << "[" << s << "]" << endl;   // [Hello]
```

### 20.4  insert：插入

`insert` 在指定位置塞入一段新文字。

```cpp
string s = "Hello World";
s.insert(5, " Beautiful");         // 在下标 5 的位置插入
cout << s << endl;                 // "Hello Beautiful World"

s.insert(0, "Say: ");              // 在开头插入
cout << s << endl;                 // "Say: Hello Beautiful World"
```

### 20.5  replace：替换

`replace` 把一段旧的换成一段新的。

```cpp
string s = "I like cats";
s.replace(2, 4, "love");           // 从下标 2 开始，删 4 个字符，放入 "love"
cout << s << endl;                 // "I love cats"

s.replace(7, 4, "dogs");           // 把 "cats" 换成 "dogs"
cout << s << endl;                 // "I love dogs"
```

格式：`s.replace(起始位置, 要删的长度, 新字符串)`——先删后插。

### 20.6  字符处理函数

这些函数处理**单个字符**（`char`），不是整个 `string`。需要 `#include <cctype>`。

| 函数 | 作用 | 示例 |
|------|------|------|
| `isdigit(c)` | c 是数字吗？ | `isdigit('5')` → true |
| `isalpha(c)` | c 是字母吗？ | `isalpha('A')` → true |
| `isupper(c)` | c 是大写吗？ | `isupper('Z')` → true |
| `islower(c)` | c 是小写吗？ | `islower('a')` → true |
| `isspace(c)` | c 是空白吗？ | `isspace(' ')` → true |
| `toupper(c)` | 转成大写 | `toupper('a')` → 'A' |
| `tolower(c)` | 转成小写 | `tolower('Z')` → 'z' |

> ❌ **常见踩坑——toupper/tolower 没赋值回去：**
>
> ```cpp
> string s = "hello";
> for (int i = 0; i < s.size(); i++) {
>     toupper(s[i]);                   // ❌ 返回了大写，但没存回 s[i]！
> }
> cout << s << endl;                   // 还是 "hello"，白干了
> ```
>
> ✅ **正确——必须把返回值存回去：**
>
> ```cpp
> for (int i = 0; i < s.size(); i++) {
>     s[i] = toupper(s[i]);            // ✅ 把转换结果赋值回去
> }
> cout << s << endl;                   // "HELLO"
> ```
>
> 🧠 `toupper(c)` 不修改 c 本身，而是**返回**转换后的新值。不用返回值 = 白调。

**实战：统计字符串中的数字个数**：

```cpp
string s = "a1b2c3d4";
int cnt = 0;
for (int i = 0; i < s.size(); i++) {
    if (isdigit(s[i])) cnt++;
}
cout << "数字个数：" << cnt << endl;  // 4
```

**实战：大小写转换**：

```cpp
string s = "Hello World";
for (int i = 0; i < s.size(); i++) {
    s[i] = toupper(s[i]);           // 全部转大写
}
cout << s << endl;                  // "HELLO WORLD"
```

> ⚠️ 别忘了 `#include <cctype>`！用了 `isdigit`/`toupper` 但没 include → 编译可能不报错但行为怪异。

### 20.7  遍历字符串的两种方式

**方式一：下标遍历**——你已经很熟悉了：

```cpp
string s = "Hello";
for (int i = 0; i < s.size(); i++) {
    cout << s[i] << " ";
}
// H e l l o
```

**方式二：范围 for 循环**——C++11 的新语法，更简洁：

```cpp
string s = "Hello";
for (char c : s) {                 // 读作：对于 s 里的每个字符 c
    cout << c << " ";
}
// H e l l o
```

> 范围 for 的"读法口诀"：`for (元素类型 变量名 : 容器)` → "对于容器里的每个元素"。

如果需要修改每个字符，用引用：

```cpp
for (char &c : s) {                // 注意 & ——引用，可以修改
    c = toupper(c);
}
cout << s << endl;                 // "HELLO"
```

两种方式各有优劣：下标遍历可以知道位置（`i`），范围 for 写法更简洁。按需选择。

用户输入一个 11 位手机号字符串，程序将中间 4 位替换为 `****` 后输出。例如输入 `13812345678`，输出 `138****5678`。（提示：用 `substr` 取头尾 + 拼接，或用 `replace` 直接替换中间部分。）

### 挑战题

**6. 编程题 — 判断回文串**

回文串就是正着读和反着读一样的字符串（如 "racecar"、"level"、"上海自来水来自海上"）。用户输入一个字符串，判断它是不是回文串。忽略大小写和非字母数字字符。

示例：输入 `"A man, a plan, a canal: Panama"` → 输出 `是回文串`

（提示：先预处理——用 `tolower` 统一小写，用 `isalnum` 过滤标点，然后双指针从两端向中间比较。）

**7. 编程题 — 压缩字符串**

实现简单行程编码（Run-Length Encoding）：对于连续重复的字符，用"字符+次数"表示。例如 `"aaabbc"` 压缩为 `"a3b2c1"`，`"abc"` 压缩为 `"a1b1c1"`。输出压缩后的字符串。

## 🧠 本章小结

```
查找  find(subs, start=0)  →  返回下标或 string::npos
截取  substr(pos, len)     →  返回截取的子串
删除  erase(pos, len)      →  删除指定段
插入  insert(pos, str)     →  在 pos 处插入
替换  replace(pos, len, str) → 删 len 个，再插入 str

字符处理（需 <cctype>）：
  isdigit/isalpha/isupper/islower/isspace
  toupper/tolower（返回转换后值，要赋值回去！）

遍历方式：
  下标 for (int i=0; i<s.size(); i++)
  范围 for (char c : s)  /  for (char &c : s)
```
