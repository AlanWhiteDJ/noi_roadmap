## 🧱 知识讲解

### 22.1  容器选择的直觉

拿到一道数据处理题，先问自己一个问题：**"数据长什么样？"**

| 数据形状 | 容器 | 典型例子 |
|----------|------|----------|
| 一列数字 | `int a[N]` | 成绩单、温度记录 |
| 一行文字 | `string s` | 姓名、文章、密码 |
| 多行同列 | `string names[N]` | 全班姓名、单词列表 |
| 行×列的表格 | `int a[N][M]` | 成绩表、棋盘、图像 |
| 表格 + 标签 | 多个数组配合 | 姓名 + 各科成绩 |

> 🎯 核心直觉：**数据的"形状"决定了容器的"形状"**。如果你发现自己在声明 5 个分散的变量来描述同一个东西，说明你需要一个数组。如果你发现数组不够用还要再加一个平行的数组，说明你可能需要二维数组或结构体。

### 22.2  经典题①：矩阵旋转

**题目**：给定一个 n×n 的矩阵，将其顺时针旋转 90 度。

**例子**：

```
输入：              输出：
1  2  3            7  4  1
4  5  6            8  5  2
7  8  9            9  6  3
```

**思路**：观察规律——旋转后，原来的第 i 行第 j 列，变成了新矩阵的第 j 行第 (n-1-i) 列。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int a[100][100], b[100][100];

    // 读入原矩阵
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            cin >> a[i][j];

    // 旋转：a[i][j] → b[j][n-1-i]
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            b[j][n - 1 - i] = a[i][j];

    // 输出新矩阵
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++)
            cout << b[i][j] << " ";
        cout << endl;
    }
    return 0;
}
```

> ❌ **常见踩坑——旋转方向搞反：**
>
> ```cpp
> b[i][j] = a[j][n-1-i];  // ❌ 这是逆时针旋转！
> ```
>
> ✅ **顺时针正确公式**：`b[j][n-1-i] = a[i][j]`——原矩阵的 (i,j) 到新矩阵的 (j, n-1-i)。
>
> 🧠 验证方法：拿一个 2×2 的小矩阵在纸上画一遍，确认公式正确再写代码。

> 💡 旋转 180° 就是转两次 90°，旋转 270° 就是转三次。掌握一个方向的变换规律就行！

### 22.3  经典题②：螺旋填数

**题目**：给定 n，按顺时针螺旋填入 1 到 n²。

**例子**（n=4）：

```
 1  2  3  4
12 13 14  5
11 16 15  6
10  9  8  7
```

**思路**：模拟"走迷宫"——维护当前坐标 (x, y) 和方向，碰到边界或已填的格子就转弯。

```cpp
int n;
cin >> n;
int a[20][20] = {0};

int dx[] = {0, 1, 0, -1};  // 右、下、左、上（x变化）
int dy[] = {1, 0, -1, 0};  // 右、下、左、上（y变化）
int dir = 0;                // 当前方向：0=右 1=下 2=左 3=上
int x = 0, y = 0;          // 当前位置

for (int k = 1; k <= n * n; k++) {
    a[x][y] = k;
    int nx = x + dx[dir];
    int ny = y + dy[dir];
    // 如果下一步出界或已经填过，就转弯
    if (nx < 0 || nx >= n || ny < 0 || ny >= n || a[nx][ny] != 0) {
        dir = (dir + 1) % 4;   // 顺时针转 90°
        nx = x + dx[dir];
        ny = y + dy[dir];
    }
    x = nx;
    y = ny;
}

// 输出矩阵
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++)
        printf("%3d", a[i][j]);  // 占 3 格对齐
    cout << endl;
}
```

> 🧩 螺旋填数是"模拟 + 数组"的经典结合。方向数组 `dx[]` 和 `dy[]` 是一种高级技巧——用循环代替四个方向的 `if-else`，代码更简洁优雅。

### 22.4  经典题③：字符串统计

**题目**：输入一段英文（可能包含空格和标点），统计其中有多少个字母、多少个数字、多少个单词。

```cpp
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

int main() {
    string line;
    getline(cin, line);  // 读入整行（含空格）

    int letters = 0, digits = 0, words = 0;
    bool inWord = false;   // 标记：当前是否在单词内部

    for (char ch : line) {
        if (isalpha(ch)) letters++;
        if (isdigit(ch)) digits++;

        if (isalpha(ch) && !inWord) {  // 进入一个新单词
            words++;
            inWord = true;
        }
        if (!isalpha(ch) && !isdigit(ch)) {  // 遇到分隔符，离开单词
            inWord = false;
        }
    }

    cout << "字母数：" << letters << endl;
    cout << "数字数：" << digits << endl;
    cout << "单词数：" << words << endl;
    return 0;
}
```

> 💡 用 `bool inWord` 标记"是否在单词内"是统计单词数的经典技巧——只在从"不在单词内"切换到"在单词内"的瞬间 +1。这样无论一个单词有多长，都只计一次。

> ❌ **常见踩坑——用空格数来统计单词：**
>
> ```cpp
> int words = 1;
> for (char ch : line) {
>     if (ch == ' ') words++;    // ❌ "Hello,  world" 两个空格会多计
> }                               //    开头有空格的段落也会错
> ```
>
> ✅ **正确——用 inWord 状态标记法**，只在新单词的起始边界 +1。

### 22.5  经典题④：成绩排名

**题目**：输入 n 个学生的姓名和三科成绩，按总分降序排名。

**关键技巧**：排序时，交换总分的同时也要**同步交换**姓名和成绩——这叫做"捆绑交换"。

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    int n;
    cin >> n;
    string name[100];
    int score[100][3];  // 语文、数学、英语
    int total[100];

    // 读入并计算总分
    for (int i = 0; i < n; i++) {
        cin >> name[i];
        total[i] = 0;
        for (int j = 0; j < 3; j++) {
            cin >> score[i][j];
            total[i] += score[i][j];
        }
    }

    // 冒泡排序（按总分降序）
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (total[j] < total[j + 1]) {
                // 捆绑交换——三步同步
                swap(total[j], total[j + 1]);
                swap(name[j], name[j + 1]);
                for (int k = 0; k < 3; k++)
                    swap(score[j][k], score[j + 1][k]);
            }
        }
    }

    // 输出排名
    cout << "排名\t姓名\t语文\t数学\t英语\t总分" << endl;
    for (int i = 0; i < n; i++) {
        cout << i + 1 << "\t" << name[i];
        for (int j = 0; j < 3; j++)
            cout << "\t" << score[i][j];
        cout << "\t" << total[i] << endl;
    }
    return 0;
}
```

> 🎯 捆绑交换是综合题的灵魂——多个平行的数组描述同一组对象的多个属性时，对任何一个属性做排序，其他属性都必须跟着一起动。

> ❌ **经典踩坑——排序时忘了捆绑交换：**
>
> ```cpp
> if (total[j] < total[j+1]) {
>     swap(total[j], total[j+1]);   // ❌ 只交换了总分！
> }
> // 结果：总分排名正确，但姓名还留在原位置——张三的成绩挂在了李四的名字下！
> ```
>
> ✅ **正确——所有相关数组同步交换：**
>
> ```cpp
> if (total[j] < total[j+1]) {
>     swap(total[j], total[j+1]);        // ✅ 总分
>     swap(name[j], name[j+1]);          // ✅ 姓名
>     swap(score[j][0], score[j+1][0]);  // ✅ 各科成绩都要换
>     swap(score[j][1], score[j+1][1]);
>     swap(score[j][2], score[j+1][2]);
> }
> ```
>
> 🧠 记忆：一条绳上的蚂蚱——动一个，全都要跟着动。

### 22.6  经典题⑤：数据去重与排序

**题目**：输入 n 个整数（可能重复），输出去重后升序排列的结果。

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n, a[1000];
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];

    // 先排序
    sort(a, a + n);

    // 去重覆盖（双指针法）
    int pos = 0;  // pos 指向"不重复区间的最后一个位置"
    for (int i = 1; i < n; i++) {
        if (a[i] != a[pos]) {
            pos++;
            a[pos] = a[i];  // 把新元素覆盖到去重区
        }
    }

    // 输出去重后的结果（pos+1 个元素）
    for (int i = 0; i <= pos; i++)
        cout << a[i] << " ";
    cout << endl;
    return 0;
}
```

> 🧩 双指针去重比"读到新数组"更省空间。这种"原地修改"的思路（不新开数组，在原数组上操作）是竞赛里的常用优化。

## 🦶 你踩过这些坑吗？

- [ ] **坑 1**：排序时只交换了总分，忘了交换姓名和成绩——张冠李戴，数据全乱了
- [ ] **坑 2**：螺旋填数中坐标更新顺序错——先填数再检查边界，最后一个数可能填在界外
- [ ] **坑 3**：单词统计用空格计数——多个空格、"Hello, world!" 这种场景会统计错误
- [ ] **坑 4**：矩阵旋转坐标公式写反——顺时针和逆时针的公式不一样，写反了方向就错了
- [ ] **坑 5**：去重时直接在原数组上一边遍历一边删除——下标全乱，比不删还乱

  一列数      → int a[N]

  数据去重   → 先排序 + 双指针原地覆盖

【关键技巧】
  捆绑交换 | 方向数组 | 状态标记 | 双指针去重
