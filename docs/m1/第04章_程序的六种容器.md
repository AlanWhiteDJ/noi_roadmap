## 📖 故事引入

小刚写了一个程序，算全校有多少学生。他把人数存在一个变量里，数字是 1280——完全正常。

然后他又写了一个程序，算他玩的游戏里一共打了多少分。分数是 8500000000（85 亿）。

程序运行……显示的结果是 **5237120**。完全不对。

小刚懵了。"我没写错啊？"

他没写错算法。他选错了"数字的容器"——有些容器只能装小数字，硬塞大数字进去就会**溢出**，像一个小杯子接一整桶水，多的全洒了。

今天我们就来搞清楚：C++ 里有哪些"容器"，分别用来装什么。而这一切的起点，就是你第 3 章已经掌握的 `int`——现在让我们看看它的五个"兄弟姐妹"。

## 🧱 知识讲解

### 4.1  数据类型是什么

计算机的内存不是无限大的。当你写下数据，计算机需要在内存里给它分配空间。**数据类型就是告诉计算机："请给我准备多大的空间、用来存什么"**。

就像你去超市买饮料，你得告诉店员："我要小杯 / 中杯 / 大杯"。不同的杯能装不同量的饮料。数据类型就是杯子的尺寸。

C++ 有六种最常用的类型，按"装的东西"分成三组：

| 分组 | 类型 | 用来装什么 |
|------|------|-----------|
| 数字 | `int` / `long long` / `double` | 整数 / 大整数 / 小数 |
| 文字 | `char` / `string` | 单个字符 / 一串文字 |
| 判断 | `bool` | 真或假 |

下面一个一个来。

---

### 4.2  整数：`int` 和 `long long`

| 类型 | 占用空间 | 能表示的范围 | 适合场景 |
|------|----------|-------------|----------|
| `int` | 4 字节 | 约 -21 亿 ~ 21 亿 | **最常用，主力** |
| `long long` | 8 字节 | 约 ±9×10¹⁸ | 数字超过 21 亿时用 |

> 记住：**`int` 能装到约 21 亿。超过这个数，立刻换 `long long`。**

```cpp
int age = 12;                     // 年龄，int 绰绰有余
long long stars = 100000000000LL; // 1000 亿颗星星，必须用 long long
```

### 4.3  什么是溢出

如果给 `int` 存一个超过 21 亿的数——

```cpp
// ❌ 错误：30 亿超过 int 上限，会溢出！
int a = 3000000000;  // 30 亿
cout << a;            // 输出：-1294967296（一个奇怪的数字）

// ✅ 正确：用 long long
long long a = 3000000000LL;
cout << a;            // 输出：3000000000
```

这叫做**溢出**——数字太大，容器装不下，多出来的部分"溢"成了乱七八糟的数字。

> ⚠️ 溢出不会报错！程序会开开心心地给你一个错误结果。这是溢出的可怕之处——你完全不知道算错了。

### 4.4  浮点数：`double`

整数用 `int`，那 3.14 怎么办？用 `double`（双精度浮点数）：

| 类型 | 占用空间 | 精度 | 适合场景 |
|------|----------|------|----------|
| `double` | 8 字节 | 约 15 位有效数字 | **所有需要小数的计算** |

```cpp
double pi = 3.14159265358979;
double score = 95.5;
double price = 12.8;
```

> 竞赛中需要小数时，一律用 `double`。

### 4.5  字符：`char`

有时候你只需要存**一个**字母或符号——比如考试成绩的等级（A/B/C/D），或者一个标点符号。这时候 `int` 不合适，`string` 又太大了。

`char` 是专门存单个字符的类型，用**单引号**括起来：

```cpp
char grade = 'A';       // 成绩等级
char firstLetter = 'Z'; // 首字母
char symbol = '?';      // 标点符号
```

`char` 占 1 个字节。它本质上存的是一个整数（字符的 ASCII 码），所以 `'A'` 在计算机内部其实是数字 65。但现阶段你把它当"装单个字符的容器"用就行。

> 💡 单引号 `'A'` 是 char，双引号 `"A"` 是 string。一个字符的区别，却完全是两种类型。

### 4.6  字符串：`string`

如果你想存一个名字、一句话、一整段文字——用 `string`。它用**双引号**：

```cpp
string name = "小明";
string greeting = "Hello, World!";
string school = "阳光小学";
```

`string` 和之前学的类型有一个重要区别：它不是 C++ 的"原生"类型，而是标准库提供的工具。使用时需要在文件开头 `#include <string>`：

```cpp
#include <iostream>
#include <string>       // ← 用 string 必须加这行
using namespace std;

int main() {
    string name = "小明";
    cout << "你好，" << name << "!" << endl;
    return 0;
}
```

> 现阶段知道怎么声明 string、怎么输出就够了。string 还有很多操作（拼接、长度、查找），后面专门有章节讲。

### 4.7  布尔值：`bool`

生活中很多问题的答案只有两种——是或不是、对或错、真或假。编程里也一样。

`bool` 类型只能存两个值：`true`（真）或 `false`（假）。

```cpp
bool isPassed = true;     // 及格了
bool isRaining = false;   // 没下雨
bool hasKey = true;       // 有钥匙
```

`bool` 占 1 个字节。它看起来简单，但以后你会频繁用到它——每个 `if` 判断、每个循环条件，背后都是 bool 在起作用。

### 4.8  sizeof：看看类型有多大

你可以用 `sizeof` 关键字来查一个类型占几个字节：

```cpp
cout << "int: " << sizeof(int) << " 字节" << endl;             // 4
cout << "long long: " << sizeof(long long) << " 字节" << endl; // 8
cout << "double: " << sizeof(double) << " 字节" << endl;       // 8
cout << "char: " << sizeof(char) << " 字节" << endl;           // 1
cout << "bool: " << sizeof(bool) << " 字节" << endl;           // 1
```

### 4.9  类型选择速查表

| 场景 | 用什么 | 示例 |
|------|--------|------|
| 计数、人数、年龄 | `int` | `int count = 42;` |
| 可能很大的数（乘积、大数据） | `long long` | `long long total = 0;` |
| 带小数（面积、平均分、价格） | `double` | `double avg = 91.5;` |
| 单个字符（等级、符号） | `char` | `char grade = 'A';` |
| 名字、一句话、文字 | `string` | `string name = "小明";` |
| 是非判断（对错、真假） | `bool` | `bool ok = true;` |

## ⚠️ 你踩过这些坑吗

- [ ] **坑 1：忘记溢出，所有数字都用 int** — `int total = 100000 * 100000;` 结果 100 亿，int 装不下。用 `long long total = 100000LL * 100000;`。
- [ ] **坑 2：整数除法当小数用** — `int a = 10 / 3;` 得到 3，不是 3.333。用 `double a = 10.0 / 3;`。
- [ ] **坑 3：char 和 string 搞混** — 单引号→char（一个字符），双引号→string（一串字符）。`char name = "小明";` 是错的！
- [ ] **坑 4：用 string 忘了 `#include <string>`** — 代码里用了 `string name;` 但没引入头文件→编译错误。开头加上 `#include <string>`。

## 🧠 本章小结

```
数字类：
  int       — 整数主力（4 字节，约 ±21 亿）
  long long — 大整数（8 字节，约 ±9×10¹⁸）
  double    — 浮点数（8 字节，15 位精度）

文字类：
  char   — 单个字符（1 字节，单引号 'A'）
  string — 一串文字（双引号 "Hello"，需 #include <string>）

判断类：
  bool   — 真/假（true / false）

溢出：超出容器范围 → 结果错误且不报错
```
