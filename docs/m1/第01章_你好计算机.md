## 🧱 知识讲解

### 1.1  计算机是什么：一个"听话的机器"

计算机本质上是一个**能执行指令的机器**。它自己不会思考，不会"想做什么就做什么"。你给它一条指令，它执行一条；给它一百条，它执行一百条。而且执行得飞快。

就像一只训练有素的狗——你扔飞盘，它叼回来；你说"坐下"，它坐下。但它不会自己决定"今天心情好，我要主动给你泡杯咖啡"。

计算机也一样。你让它算 3 + 5，它就算 3 + 5。你不让它算，它什么都不做。

那么问题来了：**你怎么"告诉"计算机做什么？**

### 1.2  计算机的五个部分

任何一台计算机——不管是台式机、笔记本，还是你口袋里的手机——都由五个部分组成：

| 部分 | 作用 | 类比 |
|------|------|------|
| **输入设备** | 把外界信息送进计算机 | 眼睛和耳朵 |
| **输出设备** | 把计算结果展示给人看 | 嘴巴和手 |
| **运算器** | 做算术和逻辑判断 | 大脑的计算区 |
| **控制器** | 指挥其他部件协调工作 | 大脑的指挥区 |
| **存储器** | 记住数据和指令 | 大脑的记忆区 |

**输入设备**：键盘、鼠标、触摸屏、麦克风、摄像头

**输出设备**：显示器、打印机、音箱

**运算器 + 控制器**合在一起就是 **CPU**（中央处理器），它是计算机的"大脑"。

**存储器**分两种：
- **内存**（RAM）：电一断，里面的东西就消失——像你在草稿纸上演算，下课铃一响就把草稿纸扔了
- **外存**（硬盘/U盘）：断电也不丢——像你的笔记本，合上了明天打开还在

### 1.3  什么是程序

"程序"这个词听起来很高级，其实就一句话：

> **程序 = 给计算机的指令清单**

你在奶茶店点单："一杯珍珠奶茶，去冰，三分糖。"这是给人看的点单指令。

程序就是给计算机看的"点单指令"——只不过计算机比奶茶店员死板一万倍。你对店员说"一杯珍珠奶茶"，他能理解你要的是奶茶不是咖啡。但计算机不行——你必须一条一条说清楚，不能有任何模糊。

举个具体的例子。假设你要写一个"算加法"的程序：

```
1. 告诉计算机：准备接收第一个数字
2. 告诉计算机：准备接收第二个数字
3. 告诉计算机：把两个数字加起来
4. 告诉计算机：把结果给我看
```

这就是一个程序。只是四条指令。

### 1.4  编程语言：人和计算机之间的翻译

你可能会问：那我用中文写"把两个数字加起来"，计算机能看懂吗？

**不能。**

计算机只认识一种语言：**机器语言**。那是一串串 0 和 1，长这样：

```
10110000 01100001
```

让人类写这种东西，一天写不了 10 行还不一定对。

于是聪明的人类发明了**编程语言**——一种介于"人话"和"机器语言"之间的语言。它长得像英语，但比英语简单得多，而且每条语句的意思都确定无疑，不会产生歧义。

比如在 C++ 里，你要输出一个句子，就写：

```cpp
cout << "Hello, World!";
```

这行代码的意思是："把 Hello, World! 这几个字送到屏幕上"。

世界上有很多种编程语言：C++、Python、Java、JavaScript……它们就像不同的"方言"。但**对信息学竞赛来说，C++ 是唯一的主角**。因为它跑得快、功能强大，几乎所有竞赛都支持。

### 1.5  程序的运行过程：IPO 模型

从你写好程序到看到结果，中间发生了什么？

用三个字母就能概括：**IPO**

| 字母 | 含义 | 例子 |
|------|------|------|
| **I** | Input（输入） | 你在键盘上敲了一个数字 5 |
| **P** | Process（处理） | CPU 把 5 存起来，等会儿用 |
| **O** | Output（输出） | 屏幕上显示"结果是 5" |

你以后写的每一个程序，本质上都是这三个步骤：

1. 接收输入
2. 做点什么
3. 给出输出

听起来简单吧？等到了后面几章，Process 那一步会变得越来越复杂。但骨架永远是这个骨架——IPO。

## ⚠️ 你踩过这些坑吗

- [ ] **坑 1：以为计算机"聪明"** — "计算机什么都会，它肯定能自动理解我的意思。" 真相：计算机是最笨的东西，它只会做你让它做的——一条都不多，一条都不少。
- [ ] **坑 2：把"编程"想得太神秘** — "编程是黑客戴着兜帽在黑暗里敲键盘。" 真相：编程就是写指令清单，跟写菜谱、写组装说明书没有本质区别。
- [ ] **坑 3：混淆"内存"和"外存"** — "内存就是 C 盘 D 盘嘛。" 真相：内存 = 工作台（关机消失）；外存 = 仓库（永久保存）。打开文件是搬进工作台，保存是搬回仓库。
