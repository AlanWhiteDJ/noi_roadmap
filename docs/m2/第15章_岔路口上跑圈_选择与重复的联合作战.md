## 📖 故事引入

小华参加校内的编程兴趣小组。老师出了一道题："判断一个数是不是质数。"

小华心里想："质数？就是只能被 1 和自己整除的数……等等，程序怎么判断'只能'？"

他用 `if (n % 2 != 0)` 试了一下——不行，9 不是质数但也能通过。他又加了 `if (n % 3 != 0)`……加到什么时候是个头？

旁边的学长扫了一眼："你想想，怎么让程序帮你**逐个试**除数？"

"逐个试……逐个……"小华眼睛一亮——`for` 循环！从 2 试到 n-1，只要有一个能整除就不是质数。

这就是"看到问题→选择结构"的思维。不是先去想代码怎么写，而是先问自己三个问题：

1. 需不需要按条件分岔？→ 用 `if/switch`
2. 需不需要做重复的事？→ 用 `for/while`
3. 分岔里面是不是还有分岔？重复里面是不是还有重复？→ 嵌套

## 🧱 知识讲解

### 15.1  思维框架：三步选结构

每见一道题，先不写代码，按这三步走：

| 步骤 | 问题 | 答案 → 结构 |
|------|------|-------------|
| ① | 需要根据条件走不同岔路吗？ | 是 → `if/else` 或 `switch` |
| ② | 需要重复做某件事吗？ | 是 → `for`(已知次数) / `while`(未知次数) |
| ③ | 岔路里还要重复？重复里还要岔路？ | 是 → 嵌套 |

先画流程图（哪怕纸上画几个框），再翻译成代码。这个习惯帮你少走一半弯路。

### 15.2  经典题①：猜数字游戏

**题目**：程序随机生成一个 1~100 的整数，玩家每次猜一个数，程序回复"大了"或"小了"，直到猜对。

**思路分析**：
- 需要重复猜直到猜对 → `while` 循环
- 每次猜测后判断大小 → `if/else` 分支
- 结论：**循环套分支**

```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));
    int secret = rand() % 100 + 1;  // 1~100 的秘密数字
    int guess, cnt = 0;

    cout << "我想好了一个 1~100 之间的数字，你猜猜看？" << endl;

    while (true) {
        cout << "你的猜测：";
        cin >> guess;
        cnt++;

        if (guess > secret) {
            cout << "太大了！再试试。" << endl;
        } else if (guess < secret) {
            cout << "太小了！再试试。" << endl;
        } else {
            cout << "恭喜猜对！你一共猜了 " << cnt << " 次。" << endl;
            break;
        }
    }
    return 0;
}
```

> 💡 用 `cnt` 统计猜测次数——这个计数器模式在循环中极其常用。

### 15.3  经典题②：循环菜单计算器

**题目**：程序显示菜单（1.加法 2.减法 3.乘法 4.除法 0.退出），用户选择后输入两个数计算结果，算完回到菜单，选 0 退出。

**思路分析**：
- 菜单需要反复显示直到用户选 0 → `while` 循环
- 选不同数字做不同运算 → `switch` 分支
- 结论：**循环套 switch**

```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;
    double a, b;

    while (true) {
        cout << "\n=== 计算器菜单 ===" << endl;
        cout << "1. 加法  2. 减法  3. 乘法  4. 除法  0. 退出" << endl;
        cout << "请选择：";
        cin >> choice;

        if (choice == 0) {
            cout << "再见！" << endl;
            break;
        }
        if (choice < 1 || choice > 4) {
            cout << "无效选择，请重试。" << endl;
            continue;   // 跳过本次循环，重新显示菜单
        }

        cout << "输入两个数：";
        cin >> a >> b;

        switch (choice) {
            case 1: cout << a << " + " << b << " = " << a + b << endl; break;
            case 2: cout << a << " - " << b << " = " << a - b << endl; break;
            case 3: cout << a << " * " << b << " = " << a * b << endl; break;
            case 4:
                if (b == 0) cout << "错误：除数不能为 0！" << endl;
                else cout << a << " / " << b << " = " << a / b << endl;
                break;
        }
    }
    return 0;
}
```

> 🎯 这个程序融合了 M1（输入输出、运算）和 M2（分支、循环）几乎所有知识——它就是一个完整的"应用程序"雏形。

### 15.4  经典题③：打印图案

**题目**：打印一个 n 行的直角三角形：

```
*
**
***
****
```

**思路**：外层循环控制行数，内层循环控制每行打印几个星号。

```cpp
int n;
cin >> n;
for (int i = 1; i <= n; i++) {      // i：第几行
    for (int j = 1; j <= i; j++) {  // j：这行打几个 *
        cout << "*";
    }
    cout << endl;  // 行末换行
}
```

> 🧩 嵌套循环的核心规律：**外层走一行，内层跑完一轮**。你可以想象外层是时针（慢），内层是秒针（快）。

### 15.5  经典题④：水仙花数

**定义**：一个三位数等于它各位数字的立方和。如 `153 = 1³ + 5³ + 3³`。

**思路**：枚举所有三位数（100~999），拆出个位、十位、百位，判断。

```cpp
for (int n = 100; n <= 999; n++) {
    int ge  = n % 10;        // 个位
    int shi = n / 10 % 10;   // 十位
    int bai = n / 100;       // 百位
    if (ge * ge * ge + shi * shi * shi + bai * bai * bai == n) {
        cout << n << " 是水仙花数" << endl;
    }
}
```

跑一遍，你会发现四个水仙花数：153、370、371、407。

### 15.6  经典题⑤：质数判断

**题目**：判断一个大于 1 的整数 n 是不是质数。

**核心优化**：不用从 2 试到 n-1，只需试到 √n！因为如果 n 有大于 √n 的因子，必定对应一个小于 √n 的因子。

```cpp
bool isPrime = true;
if (n <= 1) isPrime = false;
for (int i = 2; i * i <= n; i++) {  // i² ≤ n，等价于 i ≤ √n
    if (n % i == 0) {
        isPrime = false;
        break;   // 找到一个因子就不必继续
    }
}
cout << (isPrime ? "是质数" : "不是质数") << endl;
```

> 💡 `i * i <= n` 替代 `i <= sqrt(n)`，避免了浮点误差，是竞赛常用写法。

### 15.7  经典题⑥：完数判断

**定义**：一个数等于它所有真因子（除了自己以外的因子）之和。如 `6 = 1 + 2 + 3`。

```cpp
int n, sum = 0;
cin >> n;
for (int i = 1; i < n; i++) {
    if (n % i == 0) sum += i;
}
if (sum == n) cout << n << " 是完数" << endl;
else cout << n << " 不是完数" << endl;
```

## ⚠️ 你踩过这些坑吗

- [ ] **坑 1：循环条件写错导致死循环**——`while (choice != 0)` 但没有在循环里改 `choice` → 程序永远停不下来。确保循环条件中的变量会改变，或者用 `break` 跳出。
- [ ] **坑 2：`break` 只跳出最近一层**——嵌套循环的内层 break 只关内层的事，外层照样跑。想全停用标记变量。
- [ ] **坑 3：嵌套循环内外变量名混淆**——内外层都用 `i` → 内层"遮蔽"外层，逻辑错乱。习惯用 `i` 外层、`j` 内层、`k` 更内层。
- [ ] **坑 4：质数判断忘了 1 不是质数**——`n = 1` 时输出"是质数"。先判断 `n <= 1` 直接返回不是质数。
