# 第 14 章  刹车与跳过——break 与 continue

---

> 🏗️ **前情回顾**：上一章你学会了循环套循环——外层管行、内层管列，九九乘法表都能打出来。但套得越深越需要控制：跑到一半想停怎么办？这章给循环装上刹车和跳过键。

## 🎯 本章目标

学完这一章，你能：

- 用 `break` 提前跳出循环——不等跑完就收工

- 区分 `break` 跳出的是哪一层循环
- 在嵌套循环中用标记变量跳出多层循环

## 📖 故事引入

### 故事一：脚崴了还跑完这圈？

体育课上，老师让你跑 3 圈。跑到第 2 圈半，你突然脚崴了。

你用 `for` 循环的逻辑想："循环条件是 `i <= 3`，现在 `i = 2`，条件还成立，我不能停……"

别傻了！脚崴了就**立刻停下找医生**，不是跑完这圈再说。编程里这就是 `break`——紧急刹车，立刻退出。

### 故事二：这圈踩线了，不算

运动会 400 米比赛。你跑到第 2 圈时踩了内圈的线——裁判吹哨："踩线！这圈成绩不算，直接跑下一圈！"

你不是退出比赛，也不是重跑这一圈，而是**跳过这圈，进入下一圈**。编程里这就是 `continue`——这轮不算，下轮继续。

## 🧱 知识讲解

### 14.1  break：紧急刹车

`break` 的作用：**立刻退出当前循环，后面还剩多少轮都不跑了**——就像脚崴了立刻停，不等跑完这圈。

```cpp
// 找 1~100 中第一个能被 7 整除又能被 5 整除的数
for (int i = 1; i <= 100; i++) {
    if (i % 7 == 0 && i % 5 == 0) {
        cout << "找到了：" << i << endl;  // 35
        break;  // 找到就停，不用再找 36~100
    }
}
```

循环本来要跑 100 轮，但找到 35 就刹车了——后面的 65 轮直接跳过。这就是 `break`：**找到就够了，不浪费力气。**

### 14.2  continue：跳过当前轮

`continue` 不是"退出循环"，而是"跳过本轮剩下的代码，直接开始下一轮"——就像踩线了这圈不算，跑下一圈。

```cpp
// 输出 1~10 中的所有奇数
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;  // 偶数就跳过，不输出
    }
    cout << i << " ";
}
// 输出：1 3 5 7 9
```

> 🧠 **break vs continue**：break 是"脚崴了，不跑了，找医生"；continue 是"踩线了，这圈不算，跑下一圈"。

### 14.3  break 只跳一层——嵌套中的陷阱

`break` 只跳出**它所在的那一层**循环。如果你在嵌套循环的内层写 break，只跳出内层，外层继续跑：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        if (j == 3) {
            break;  // 只跳出内层 j 循环
        }
        cout << "(" << i << "," << j << ") ";
    }
    cout << endl;  // 外层继续，进入下一行
}
```

输出：

```
(1,1) (1,2)
(2,1) (2,2)
(3,1) (3,2)
(4,1) (4,2)
(5,1) (5,2)
```

看！每行内层都只跑到了 `j=2`（到 3 就 break 了），但外层 5 行照样全部执行。**内层的 break 管不到外层。**

### 14.4  怎么跳出所有循环？——标记变量

如果你想在内层发现某个条件后，把外层也一起停掉，怎么办？需要一个**标记变量**（也叫"标志位"）：

```cpp
bool found = false;           // 标记：还没找到
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        if (i * j > 10) {    // 第一个乘积超过 10 的位置
            cout << "找到了：" << i << "×" << j << "=" << i*j << endl;
            found = true;    // 插旗子：找到了！
            break;           // 先跳出内层
        }
    }
    if (found) break;        // 看到旗子，外层也停！
}
// 输出：找到了：3×4=12
```

> 💡 标记变量的思路：**设一个 `bool` 变量当旗子。内层发现问题时"插旗子"然后 break，外层每轮检查旗子——看到旗子也 break。** 这是跳出多层循环最清晰的方法。

### 14.5  break 和 continue 用在 while 里

break 和 continue 不只在 for 里能用，while 和 do-while 也能用：

```cpp
// while + break：密码最多试 3 次
int attempts = 0;
string password;
while (attempts < 3) {
    cout << "请输入密码：";
    cin >> password;
    if (password == "cpp123") {
        cout << "登录成功！" << endl;
        break;            // 对了就立刻退出，不用管 attempts
    }
    attempts++;
    cout << "密码错误，还剩 " << 3 - attempts << " 次机会" << endl;
}
if (attempts == 3) {
    cout << "3 次机会用完，账户已锁定。" << endl;
}
```

```cpp
// while + continue：只处理正数，跳过负数
int n;
while (cin >> n) {
    if (n < 0) {
        continue;    // 负数跳过，不累加
    }
    sum += n;
}
```

### 14.6  break 和 continue 的使用原则

- ✅ **break**：找到第一个就够、提前退出省时间、菜单里选"退出"
- ✅ **continue**：过滤不需要的数据、跳过无效输入
- ❌ **不要滥用**：如果能用正常的循环条件表达退出/跳过，就别用 break/continue——正常的条件更容易看懂
- ❌ **continue 不要放在最后一行**：`continue` 跳过的是"它和循环末尾之间"的代码，放在最后等于啥也没跳

## ⚠️ 你踩过这些坑吗

- [ ] **坑 1：以为 break 能跳出所有循环**——在内层写 `break` 只跳出内层，外层继续跑。想跳出多层，用标记变量。
- [ ] **坑 2：continue 放到代码最后一行**——`for (...) { cout << i; continue; }` 写了等于没写，后面本来就没代码。`continue` 要放在想跳过的代码**之前**。
- [ ] **坑 3：break/continue 跟循环条件打架**——循环条件说"跑 100 次"，break 说"第 35 次就停"。如果你早就知道第 35 次停，为什么不直接写循环条件呢？能用条件表达的逻辑，别依赖 break。
- [ ] **坑 4：标记变量忘了在外层判断**——内层插了旗子但外层没检查，等于白插。别忘了在内层 break 之后，外层也要 `if (found) break;`。

## 🧠 本章小结

```
循环控制：
  break    — "脚崴了，立刻停" → 跳出当前循环
  continue — "踩线了，这圈不算" → 跳过本轮，下轮继续

嵌套中的 break：
  只跳出它所在的那一层！
  跳出多层 → bool found 标记变量：内层插旗，外层检查

使用原则：
  ✅ 找到第一个就够 → break
  ✅ 过滤不需要的数据 → continue
  ❌ 别滥用——能用循环条件表达的别用 break
  ❌ continue 别放最后——放了等于没放
```
