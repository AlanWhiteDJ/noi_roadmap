## 📖 故事引入

你有没有盯着墙上的时钟发过呆？

时钟有三根针：时针、分针、秒针。它们的运动是"嵌套"的——**秒针每走 60 圈，分针走 1 格；分针每走 60 圈，时针走 1 格**。

如果你要模拟这个时钟，你会怎么写？你会写三个循环，一个套在另一个里面：

```
对于时针从 1 到 12：
    对于分针从 0 到 59：
        对于秒针从 0 到 59：
            滴答！
```

这就是**嵌套循环**——一个循环里面还装着另一个循环。

## 🧱 知识讲解

### 13.1  什么是嵌套循环

嵌套循环就是一个循环的"身体"里面又写了一个循环。外面那个叫**外层循环**，里面那个叫**内层循环**。

想象你在上体育课。老师说："全班 4 排，每排同学依次报数 1 到 5"。你脑子里自动就会：

```
对于每一排（1 到 4）：
    对于排里的每个人（1 到 5）：
        报数！
```

写成 C++ 就是这样：

```cpp
for (int row = 1; row <= 4; row++) {
    for (int col = 1; col <= 5; col++) {
        cout << row << "排" << col << "号  ";
    }
    cout << endl;  // 一排报完，换行
}
```

输出：

```
1排1号  1排2号  1排3号  1排4号  1排5号
2排1号  2排2号  2排3号  2排4号  2排5号
3排1号  3排2号  3排3号  3排4号  3排5号
4排1号  4排2号  4排3号  4排4号  4排5号
```

> 🧠 **执行顺序**：外层循环每走一次，内层循环要完整地跑一轮。就像体育课，每换一排，那一排的 5 个人全部报完数，才轮到下一排。

### 13.2  打印矩形

学会了嵌套循环，你就可以在屏幕上"画画"了。先从最简单的矩形开始——5 行 8 列的星星：

```cpp
for (int i = 1; i <= 5; i++) {       // 控制行数
    for (int j = 1; j <= 8; j++) {   // 控制每行的列数
        cout << "*";
    }
    cout << endl;                     // 每行打印完，换行
}
```

输出：

```
********
********
********
********
********
```

> `i` 控制"画几行"，`j` 控制"每行画几个"。先想清楚哪层管行、哪层管列。

### 13.3  打印三角形

矩形太简单了。来个三角形——第一行 1 颗星，第二行 2 颗，第三行 3 颗……

关键发现：**第 i 行的星星数 = i**。

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {   // j <= i：第i行打印i颗星
        cout << "*";
    }
    cout << endl;
}
```

输出：

```
*
**
***
****
*****
```

> 🎨 内层循环的上限 `j <= i` 是动态的——它会随着外层 i 变大而变大。这就是"三角形"的秘密。

如果要打印**倒三角形**，把上限改成 `j <= 6 - i`：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 6 - i; j++) {
        cout << "*";
    }
    cout << endl;
}
```

输出：

```
*****
****
***
**
*
```

### 13.4  打印菱形

菱形 = 正三角形 + 倒三角形。先画上半部分，再画下半部分：

```cpp
int n = 4;  // 菱形一半的高度

// 上半部分（正三角）
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n - i; j++) cout << " ";   // 空格
    for (int j = 1; j <= 2 * i - 1; j++) cout << "*"; // 星星
    cout << endl;
}

// 下半部分（倒三角）
for (int i = n - 1; i >= 1; i--) {
    for (int j = 1; j <= n - i; j++) cout << " ";
    for (int j = 1; j <= 2 * i - 1; j++) cout << "*";
    cout << endl;
}
```

输出：

```
   *
  ***
 *****
*******
 *****
  ***
   *
```

### 13.5  打印空心图形

"空心"的意思是——只在边框上画星，中间留白。规律：**第一行、最后一行、第一列、最后一列才打印星**。

空心矩形：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 8; j++) {
        if (i == 1 || i == 5 || j == 1 || j == 8)
            cout << "*";
        else
            cout << " ";
    }
    cout << endl;
}
```

输出：

```
********
*      *
*      *
*      *
********
```

> `||` 是"或"运算符（第 9 章学过）。"只要满足任意一个边框条件，就打星"。

### 13.6  九九乘法表

学嵌套循环，九九乘法表是必修课。对每一对 (i, j)，输出 `i × j = 积`：

```cpp
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {        // j <= i：只打下半三角
        cout << j << "×" << i << "=" << i * j << "\t";
    }
    cout << endl;
}
```

输出：

```
1×1=1
1×2=2   2×2=4
1×3=3   2×3=6   3×3=9
1×4=4   2×4=8   3×4=12  4×4=16
...
1×9=9   2×9=18  3×9=27  4×9=36  5×9=45  6×9=54  7×9=63  8×9=72  9×9=81
```

> `\t` 是制表符（Tab），让各列自动对齐，看起来更整齐。

## ⚠️ 你踩过这些坑吗

- [ ] **坑 1：内外层变量搞混**——内层循环里写 `i` 却想操作列，或者内外层用同样的变量名。记住：外层用 `i`、内层用 `j`，这是全世界的习惯。
- [ ] **坑 2：搞错内外层换行时机**——`cout << endl;` 应该放在外层循环里、内层循环**之后**。如果放在内层里，每打印一个星就换一行，输出变成一竖条。
- [ ] **坑 3：三角形内层上限写死**——`for (int j = 1; j <= 5; j++)` 打印出来永远是矩形。三角形的关键是内层上限要用外层变量 `i`：`j <= i`。
- [ ] **坑 4：空心图形的条件写反了**——把"什么时候该打星"写成了"什么时候不该打"。想清楚边框是 `i==1 || i==n || j==1 || j==m`。

## 🧠 本章小结

```
嵌套循环：
  外层（行）─── for (int i = ...)
  内层（列）─── for (int j = ...)   ← 外层每走一步，内层跑完整轮

经典应用：
  矩形    → 五行八列，内层 j<=定值
  三角形  → 第i行打i个，内层 j<=i
  倒三角  → 内层 j<=n+1-i 或 j<=6-i
  菱形    → 正三角 + 倒三角，两段组合
  空心    → 边框用 if 判断 (i==1 || i==n || j==1 || j==m)
  乘法表  → 两层 1~9，内层 j<=i（三角）或 j<=9（矩形）
```
